# Fit Power Quilpué — Demo comercial multicede

Landing premium + demo visual de **Fit Power OS**, pensada para venderle al cliente una primera etapa digital para sus 3 gimnasios.

## Qué incluye

- Landing pública premium y responsive.
- Hero comercial con CTA a prueba gratis y demo del sistema.
- Sección Fit Power OS para vender la plataforma multicede.
- Sedes demo: Quilpué Centro, El Belloto y Villa Alemana.
- Planes Start, Power y Elite.
- Servicios: HIIT, Funcional y Personal Training.
- Vista previa de dashboard admin multicede.
- Rutas demo para agenda, checkout y roles.

## Rutas

- `/` Landing principal.
- `/demo` Overview del sistema.
- `/demo/admin` Dashboard admin demo.
- `/demo/agenda` Agenda demo.
- `/demo/checkout` Checkout demo.
- `/demo/roles` Roles demo.

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm run start
```

## Nota

Los datos, precios y métricas son de demostración. Antes de publicar para el cliente final se deben validar sedes, precios, horarios, teléfonos, email y enlaces reales.
